# Serino Mini Project

## How to use
1. Clone the repository
2. run **npm install** on the folder directory
3. run **npx tsc** on the folder directory
4. click Launch Program using visual studio code

## Postman Details
> Endpoint : http://localhost:3000/geolocation

Payload

```
{
    "Latitude":<number>,
    "Longitude":<number>,
    "Distance":<number>,
    "Prize":<number>
}
```
